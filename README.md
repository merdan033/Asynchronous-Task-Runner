# Asynchronous Task Runner

A professional Node.js task runner demonstrating callbacks, Promises, and async/await patterns with a beautiful web interface.

## Quick Start

### 1. Start the Web Server

```bash
npm run frontend
```

Or:

```bash
node server.js
```

### 2. Open in Browser

Open your browser and navigate to:
```
http://localhost:3000
```

### 3. Use the Interface

- Click the buttons to test different async patterns
- Watch tasks process in real-time
- See results appear as tasks complete

## Command Line Version

To run the command-line version:

```bash
npm start
```

Or:

```bash
node taskRunner.js
```

## Features

- ✅ Callback-based implementation (error-first pattern)
- ✅ Promise-based implementation
- ✅ Async/await implementation
- ✅ Promise.all() for parallel processing
- ✅ Promise.race() for first-to-complete
- ✅ Custom error handling demonstration
- ✅ Beautiful web interface with real-time updates
